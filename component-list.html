BUILDING YOUR PARTY

Here you can find the component and software for building your personal party-time project. The list is intended for a 2-room party, so, if more rooms are needed, consider adding more component, except for the server.

Start building!

The server

Based on Raspberry Pi 2 or a newer one, the server is intended to control every room and store the music database and also the user's database. The main functions will be implemented on the server and so this will be our reasoning of the data received from all the other components.

CLIENT

The client

The client is to be intended as the room-hub, controlling every single light in that space, it will receive the playlist to play and will update the server about the distance between the users and the estimote location beacon.

OTHER hardware

Other Components

The estimote location beacon will detect the power of the signal of other bluetooth devices present. Then it will communicate with the client. The philips hue will give the guests a clear understanding of the kind of music present in the room. The speakers are the main device to deliver the music for the whole room.

SOFTWARE

Software

The main features are implemented by our software it will reason and interact with guests. The registration of the users will be implemented through a telegram bot and it will provide the voting functionalities described previously.
